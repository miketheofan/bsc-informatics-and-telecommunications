-> Η εντολή μεταγλώττισης του προγράμματος είναι > make compile
-> Για να εκτελεστεί το πρόγραμμα πρέπει να δωθεί η εντολή 
> java Main [file1] [file2] ... [fileN], 
όπου file1,file2,etc είναι τα αρχεία που δίνονται ως όρισμα στο πρόγραμμα με σκοπό να τους πραγματοποιηθεί semantic analysis.
-> Για να εκτελεστεί το πρόγραμμα πρέπει επίσης τα .jar αρχεία να βρίσκονται στον φάκελο ../
-> Για να υλοποιήσω το ζητούμενο της άσκησης δημιούργησα 2 visitors. Ο πρώτος μαζεύει όλους τους identifiers όπως classes,methods,fields,etc και τους εισχωρεί στο symbol table, κάνοντας τους σωστούς ελέγχους με σκοπό να πετάξει Exception εάν υπάρχει κάποιο error στο declaration κάποιου identifier, όπως για παράδειγμα να ορίζεται ένα field ενώ έχει ήδη οριστεί με άλλο type προηγουμένως ή αν μια κλάση έχει ξανά οριστεί. Ο δεύτερος πραγματοποιεί όλους τους υπόλοιπους ελέγχους, όπως να κάνουν match τα types τον identifiers που χρησιμοποιούνται σε κάποιο expression, αξιοποιώντας πάντα τα δεδομένα που υπάρχουν στο symbol table από το πρώτο visit.
-> Το symbol table το υλοποίησα αξιοποιώντας τη δομή LinkedHashMap της Java. Η ιεραρχεία των δεδομένων είναι η εξής:
	1) Το Symbol Table(ST) διατηρεί ένα LinkedHashMap που περιέχει όλες τις κλάσεις που έχουν δηλώθεί.
	2) Κάθε μια από αυτές τις κλάσεις(classIdentifier) περιέχει ένα LinkedHashMap για όλα τα fields(Field) που έχουν γίνει declared στη κλάση και ένα δεύτερο για όλα τα methods(Method) που επίσης έχουν γίνει declared στη κλάση.
	3) Κάθε Method έχει ένα LinkedHashMap για όλες τις παραμέτρους της συνάρτησης και ένα δεύτερο για όλα τα fields που έχουν δηλώθεί μέσα στη συνάρτηση.
	4) Ένα Field διατηρεί δεδομένα για το όνομα της εκάστοτε μεταβλητής καθώς και τον τύπο της.
-> Για το δεύτερο argument της συνάρτησης visit, επέλεξα να δημιουργήσω μια κλάση arguHelper η οποία χρησιμοποιείται ουσιαστικά για να παρέχει την πληροφοριά του σε ποια κλάση και ποια μέθοδο "βρίσκεται" η εκάστοτε κλήση μιας visit.
-> Έχω διατηρήσει το αρχείο Example.java που υπήρχε στο αρχείο που έχει ανέβει στην εκφώνηση της άσκησης και περίέχει ένα από τα examples που δώθηκαν με την εκφώνηση.
-> Για να μεταγλωττιστεί το πρόγραμμα και να τρέξει το Example.java έχω συμπεριλάβει την εντολή > make all (ή > make).
-> Όλος ο κώδικας περιέχει σχόλια για ευκολότερη κατανόηση των σχεδιαστικών επιλογών στην υλοποίησή μου.
