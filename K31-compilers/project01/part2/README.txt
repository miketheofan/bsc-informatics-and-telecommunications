-> Η εντολή μεταγλώττισης του προγράμματος παρέμεινε > make compile.
-> Η εντολή εκτέλεσης του προγράμματος παρέμεινε > make execute.
-> Για να εκτελεστεί το παραγόμενο Main.java αρχείο, που ουσιαστικά περιέχει το output του προγράμματος, έχω συμπεριλάβει την εντολή > make test. Διαφορετικά πρέπει να δωθούν οι εντολές > javac Main.java και > java Main.
-> Η εντολή > make myexecute εκτελεί το αρχείο χρησιμοιώντας ένα input file (input), και αποθηκεύοντας το output στο αρχείο Main.java. By default, το input file έχει το παράδειγμα #3 που δώθηκε στην εκφώνηση της άσκησης.
-> Έχω επίσης συμπεριλάβει για ευκολία την εντολή > make all, η οποία μεταγλωττίζει το πρόγραμμα και εκτελεί το πρόγραμμα.
-> Η εντολή > make myall κάνει τη παραπάνω λειτουργία αλλά εκτελεί και το Main.java με σκοπό να παραχθεί το αποτέλεσμα του εκτελέσιμου που παρήγαγε ο parser.
-> Τα .jar αρχεία για να δουλέψει το πρόγραμμα θα πρέπει να βρίσκονται στο ../ directory, ακολουθώντας το format που διατηρούταν στον .zip φάκελο που δώθηκε στη σελίδα του μαθήματος.
-> Η άσκηση "πατάει" πάνω στον κώδικα που παρουσιάστηκε στο εργαστήριο της άσκησης αλλά είναι αλλαγμένο με σκοπό να ικανοποιεί το ζητούμενο της άσκησης.
-> Το πρόγραμμα μου χρησιμοποιεί το "κόλπο" που αναφέρθηκε από τον κ.Σμαραγδάκη σε διάλεξη σχετικά με το να θεωρούμε τους χαρακτήρες ')' και '{' με σκοπό να ξεχωρίζονται πιο έυκολα τα top level calls από τα lower level calls.
